# Contributing to Cricket Match Simulator

Thank you for your interest in contributing to the Cricket Match Simulator project!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone <your-fork-url>`
3. Create a virtual environment: `python -m venv venv`
4. Activate it and install dependencies: `pip install -r requirements.txt`
5. Create a branch for your feature: `git checkout -b feature/your-feature-name`

## Code Guidelines

- Follow PEP 8 style guidelines
- Write tests for new features
- Ensure all tests pass before submitting PR
- Add documentation for new features
- Keep commits atomic and meaningful

## Running Tests

```bash
pytest tests/ -v
```

## Submitting Changes

1. Push your changes to your fork
2. Create a Pull Request with a clear description
3. Link any related issues
4. Wait for review and address feedback

## Reporting Issues

- Check if issue already exists
- Provide detailed description and steps to reproduce
- Include Python version and OS information
- Attach screenshots if relevant

Thank you for contributing! 🏏
