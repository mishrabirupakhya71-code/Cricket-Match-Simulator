## [1.0.0] - 2024-01-01

### Added
- Initial release of Cricket Match Simulator
- Desktop console simulator with ball-by-ball cricket match simulation
- Strike rotation (odd/even runs)
- All 6 wicket dismissal types
- Extras handling (wides, no-balls, leg-byes, byes)
- IPL Dream11 fantasy points calculation
- Match pause/resume functionality
- Global player rankings
- Custom SQL execution support
- Real-time leaderboards
- Special awards system
- Complete test suite
- Full documentation and setup guide

### Features
- ✅ 100% accurate cricket match simulation

- ✅ Professional UI/UX
